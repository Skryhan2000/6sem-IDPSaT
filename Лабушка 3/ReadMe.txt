1. Как наглядно представить статистику странны по местоположению?

Самый простой способ выбрать ее в панели Show Me и задать правильный
вход, например, contry и sales.

2. Какие два способа экспорта данных вы знаете?

Excel: клик ПКМ на графике -> Copy ->Data, затем вставка на новый 
лист.xxls
Возможности Tableau: клик ПКМ на текущую вкладку и выбрать “Duplicate 
as crosstab”

3. Линии тренда?

Линии тренда — элемент аппарата технического анализа, используемый 
для выявления тенденций изменения данных различных элементов.
Если требуется установить связь элементов строится она в режиме Liner.
На основе получившейся информации можно сделать вывод. (Величина R^2,
значение пропорцианально влиянию)

4. Как провести анализ данных в конкретном райони страны на основе
карты?

Используя элементы выделение, например, ласо, определяем требуемы район.
Используя скрепку связываем данные. Получаем результат группировки,
например, элемент contry(group). В нем есть все данные по этому району.

5. С помощью какой кнопки можно настраивать отображение графиков?

Кнока Marks. Это кластер: color (настройкки цвета), 
                          size (размера объекта),
                          label (текста), 
                          delait (деталей), 
                          tooltip (дрейф отсутсвующих данных с др. 
                                   графика).

6. Каким цветом отображаются функции/объекты?

Зеленый/синий

7. Какая функция у команды Single Value list?

У нас есть сборный объект, если мы хотим фильтровать по одному 
признаку такие объеты, мы используем эту функцию.
